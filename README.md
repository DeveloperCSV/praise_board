## 注意事项

使用版本2.3的时候students_name.txt应该另存为ANSI编码，2.4版本应该另存为UTF-8编码，不然会报错

主分支内的students_name.txt为UTF-8编码

在IDEL运行时执行退出操作后不会退出，但在其他编辑器内是可以正常使用的
